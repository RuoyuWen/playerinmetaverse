📋 项目简介

Inner Child Chatbot 是一个专业的心理治疗辅助工具，帮助用户通过AI技术进行内在小孩疗愈。 系统通过分析用户的童年记忆问卷和给童年自己的信件，生成个性化的童年人格画像，并提供治疗性对话。

🧠 智能分析

基于GPT-4的文档分析，深度解读童年记忆和情感表达

👤 人格画像

结合MBTI理论，生成个性化的童年人格画像

💬 治疗对话

与AI化身的童年自己进行温暖的治疗性对话

📊 会话管理

完整的会话记录和治疗进程跟踪

🔧 主要功能

核心特性

  • 📝 文档分析：上传童年记忆问卷和信件，AI自动分析提取关键信息
  • 🧠 MBTI集成：支持16种MBTI人格类型，生成个性化童年画像
  • 💬 治疗对话：与AI化身的童年自己进行治疗性对话
  • 📊 会话管理：保存和管理治疗会话记录
  • 🔒 隐私保护：本地部署，数据安全可控

API接口

GET / # 服务状态 GET /api/mbti-types # 获取MBTI类型列表 POST /api/upload-documents # 上传分析文档 POST /api/chat # 聊天对话 GET /api/session/{session_id} # 获取会话信息 GET /docs # API文档

🚀 本地安装指南

步骤 1: 克隆项目

git clone https://github.com/RuoyuWen/playerinmetaverse.git cd playerinmetaverse

步骤 2: 安装依赖

pip install -r backend/requirements_simple.txt

步骤 3: 配置API密钥

在环境变量中设置您的OpenAI API密钥：

export OPENAI_API_KEY="your-api-key-here"

步骤 4: 启动服务

python start.py

步骤 5: 访问应用

打开浏览器访问显示的本地地址（通常是 http://localhost:8001）
