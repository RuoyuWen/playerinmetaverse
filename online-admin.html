Setup Instructions (One-time)

To enable online configuration, you need to set up a free cloud storage account:

  1. Visit JSONBin.io and create a free account
  2. Create a new "Bin" (JSON storage) for your configuration
  3. Copy your Bin ID and API Key
  4. Update the online-global-config.js file with your credentials:
    configEndpoint: 'https://api.jsonbin.io/v3/b/YOUR_BIN_ID'
    apiKey: '$2a$10$YOUR_API_KEY'
  5. Deploy the updated file to your server
  6. Refresh this page and start using online configuration!

Alternative: You can also use Firebase, MongoDB Atlas, or any other cloud JSON storage service.
Results Storage: Chat results are automatically saved to a separate cloud storage bin for admin access.
